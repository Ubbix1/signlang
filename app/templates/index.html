API Integration

For Developers

SignAI provides a comprehensive RESTful API for integration with your applications. Our API endpoints allow you to:

  • Authenticate users securely with JWT
  • Send hand landmark data for prediction
  • Receive real-time sign language translations
  • Access user history and other user data

All endpoints are secured and follow best practices for API development.

Example API Request 

POST /api/prediction/predict
Content-Type: application/json
Authorization: Bearer <your_token>

{
  "landmarks": [
    [0.1, 0.2], [0.3, 0.4], ...
  ],
  "save_result": true
}
